# Money Manager Application

## Description
This is a simple Money Manager application built using HTML, JavaScript and Node.js.
It allows users to add income and expenses, view total balance, and see transaction history.

## Features
- Add Income
- Add Expenses
- View Balance
- View Transactions
- Persistent data storage using JSON file

## Technologies Used
- HTML
- JavaScript
- Node.js
- Express.js

## Project Structure
- server.js – Backend server
- test.html – Frontend UI
- data.json – Stores income and expense data
- package.json – Project dependencies
